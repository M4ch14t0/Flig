# 💳 Pagamento - Flig MVP

Esta pasta contém os ícones dos métodos de pagamento aceitos pela Flig.

## 📋 Métodos de Pagamento

### Cartões de Crédito:
- `mastercard.png` - Mastercard
- `visa.png` - Visa
- `elo.png` - Elo
- `amex.png` - American Express
- `hipercard.png` - Hipercard

### Outros Métodos:
- `pix.png` - PIX
- `boleto.png` - Boleto bancário
- `paypal.png` - PayPal
- `pagseguro.png` - PagSeguro

## 🎨 Especificações

- **Formato**: PNG (com transparência)
- **Tamanho**: 60x40px (proporção 3:2)
- **Fundo**: Transparente
- **Estilo**: Oficial das marcas
- **Cores**: Originais das marcas

## 📝 Exemplo de Uso

```jsx
// Métodos de pagamento
<div className="payment-methods">
  <img src="/assets/payment/visa.png" alt="Visa" />
  <img src="/assets/payment/mastercard.png" alt="Mastercard" />
  <img src="/assets/payment/elo.png" alt="Elo" />
  <img src="/assets/payment/amex.png" alt="American Express" />
  <img src="/assets/payment/hipercard.png" alt="Hipercard" />
</div>
```
